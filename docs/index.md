# 📚 transcribe-assistant — Documentation

Welcome to the official documentation for **transcribe-assistant**, a low-latency (~250 ms) offline speech-to-text engine.

---

## 📘 Description

transcribe-assistant is an offline, low-latency (~250 ms) real-time speech-to-text engine designed for automation workflows, assistants, accessibility tools and privacy‑sensitive environments.  
It runs entirely on-device using local Faster‑Whisper inference, with zero cloud dependencies, zero telemetry, modular backend architecture, session recording, and a deterministic PRO version for compliance‑ready and reproducible use cases.
